## Overview

This repository is public so the community can access:

- Documentation  
- Issues  
- Discussions  
- Application forms  

However, the repository itself is **protected against unauthorized changes**.

## Branch Protection

The `main` branch uses strict protection rules:

- Pull requests are required  
- Code Owner review is mandatory  
- Direct pushes are blocked  
- Force pushes are blocked  
- Bypass permissions are disabled  
- Only one merge method (squash) is available  

These rules ensure that **no external PR or push can modify the repository**.

## Code Owners

All files are owned by:
@diskawrs

This means only the owner can approve or merge changes.

## Wiki Security

The wiki is public to read but locked to editing:

- Only collaborators can modify the wiki  
- No outside user can change documentation  

## Actions Security

GitHub Actions are restricted to:

- Collaborators only  
- Verified workflows  
- Secrets (webhooks, tokens) are stored securely

## Reporting Issues

Please report bugs or feedback through GitHub Issues.

## Responsible Disclosure (Optional)

This repository contains no sensitive code or infrastructure.
